# space-tavern

# Setup

0. install python3 and pip
1. `pip install -r requirements.txt`
2. python3 src

# Feature list

* Turn-based
* Single inventory slot
* Use or throw held item
* Only one use items
* Evil aliens
* Transport Beer from planet to planet for money
* Use money to buy:
  * Planet's Beer
  * Information
    * enemy types
    * spawn areas
  * Gadgets
* Background space panorama\*

# Gadget List

* Plasma gun
  * Shoots projectile dealing 1 damage
  * 4 ticks to reload
* Jump pad block
  * Stepping on the pad will launch you up in the air
* Turret
  * Shoots enemy when in sight
* Za Warudo
  * Stops time for 8 ticks

# Alien List

* Basic Alien
  * Move 4 ticks
  * Move 1 tile
  * Jump 3 tile
  * Gravity 1 tile
  * Projectile:
    * No Gravity
    * 1 damage
